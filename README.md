# 420BloomTools
Project Idea:

“The Automagic Gardener”A toolkit for hobbyists and DIY enthusiasts that bridges hardware (like IoT devices) with code for real-world applications.

Focus Areas:

1. Microgreen Monitor: Code and setup to monitor and automate care for indoor plants using low-cost Arduino or Raspberry Pi sensors.

2. Wax-on, Wax-Off Automations: Gamify your daily chores—automate reminders like watering, calendar-based tasks, or self-quirky timers like debugging “miTch errors-only analog nags.”
